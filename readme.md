# Free Online Braille Translator

![Braille Translator Screenshot](screenshot.png)

---

**Instant. Accurate. 100% Free.**  
UEB Grade 1 & 2 · No signup · Works offline · Download .BRF

Live at: [https://brailletranslators.com](https://brailletranslators.com)

---

### Features
- Real-time English ↔ Braille translation
- Full UEB compliance
- Grade 1 (letter-by-letter) & Grade 2 (contractions)
- Export as `.BRF` (embosser-ready) or `.TXT`
- No tracking · No data stored

---

### How to Use
1. Go to [brailletranslators.com](https://brailletranslators.com)
2. Type or paste text
3. Click **Translate**
4. Copy or download

---

### Tech
- Pure HTML + CSS + JavaScript
- Runs 100% in your browser
- No backend, no cookies, no logs
